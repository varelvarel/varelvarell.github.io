Carding With Termux Part 1
Nah, kali ini gw bakal kasih materi soal cara dapetin cc buat carding.\
Iya carding, gw tau jiwa miskin kalian meronta2.
Nah kali ini kita gunakan hp aja... Bisa juga pake PC/laptop.

Bahan: - Hp, Pc, atau laptop
- Termux (buat di hp)
- Internet
- Tools Sqlmap

Okey, pertama kalian Install Termux Di Playstore atau Google.
Usahakan Memori masih agak lega, se kentang2nya hp kalian, paling nggak ada memori free.
Nah, setelahnya buka termux, jangan buka baju. Lu kira mo mandi? :v
Nah ketik di termux :

$ pkg update && upgrade
Klik enter
Itu buat ngupdate paket nya
Nah setelah nya...

$ pkg install python2
$ pkg install git $ git clone https://github.com/sqlmapproject/sqlmap.git
$ cd sqlmap
$ python2 sqlmap.py

Nah, inget setiap nulis 1 kalimat script selalu di klik enter. Misal "pkg install python" Terus enter.
Nah buat yg di pc bisa install python 2.7 terus download sqlmap.
Nanti di jalanin di cmd. Atau kalo mau lebih keren install kali linux dual boot, atau pake virtualbox.
