How to install nethunter on termux

Instal Kali NetHunter di perangkat Android apa pun yang tidak di-root tanpa membatalkan garansi.

cara install :

kali@kali:~$ termux-setup-storage
kali@kali:~$ pkg install wget
kali@kali:~$ wget -O install-nethunter-termux https://offs.ec/2MceZWr
kali@kali:~$ chmod +x install-nethunter-termux
kali@kali:~$ ./install-nethunter-termux

use :

$ nethunter

Tips :

Jalankan Sudo apt update && Sudo apt full-upgrade -y hal pertama setelah instalasi untuk memperbarui Kali.
Jika Anda memiliki banyak ruang penyimpanan, Anda mungkin ingin menjalankan Sudo apt install -y kali-linux-default juga.

Beberapa utilitas seperti "atas" tidak akan berjalan pada ponsel yang tidak di-root.

Pengguna non-root masih memiliki akses root di chroot. Itu hal yang pro. Sadarilah itu.
Ponsel Galaxy dapat mencegah pengguna non-root menggunakan sudo.
Cukup gunakan "su -c" sebagai gantinya.
Lakukan pencadangan rutin rootf Anda dengan menghentikan semua sesi nethunter dan ketik berikut ini dalam sesi termux: tar -cJf kali-arm64.tar.xz kali-arm64 && mv kali-arm64.tar.xz storage/downloads Itu akan menempatkan cadangan di folder unduhan Android Anda.

Catatan: pada perangkat yang lebih lama, ubah "arm64" menjadi "armhf"
